## Early Exit Vault
When arbitrageurs buy Yes shares on one prediction market and No shares on another similar prediction market for less than one dollar, they normally have to wait until both markets expire to realize a profit. Using this vault, they can get it instantly.


## Usage

### Build

```shell
$ npm install
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```
